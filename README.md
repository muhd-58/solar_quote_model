# Solar Quote Model

Enhanced Solar Power System Sizing Model with Machine Learning and Uncertainty Quantification.

## Overview
This project provides a data-driven framework for estimating solar power system sizes. It integrates machine learning techniques with uncertainty quantification to deliver more reliable solar quotes.

## Features
- ML-based solar system sizing
- Uncertainty quantification for robust predictions
- Modular and extensible codebase
- Easy-to-use interface for solar quote generation
